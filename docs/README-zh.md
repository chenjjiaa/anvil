# Anvil 项目文档（中文版）

欢迎来到 Anvil 项目文档！本文档提供了 Anvil 区块链撮合引擎系统的详细说明。

## 📚 文档目录

### 1. [快速入门指南](quick-start-zh.md)

适合初次接触 Anvil 的开发者，包含：

- 项目简介和核心特性
- 快速开始步骤
- 核心概念说明
- 使用示例
- 常见问题解答

**推荐阅读顺序**：如果您是第一次使用 Anvil，请从这里开始。

### 2. [架构指南](architecture-guide-zh.md)

详细的系统架构和设计说明，包含：

- 项目概述和设计原则
- 整体系统架构
- 核心模块详解：
  - Gateway（订单网关）
  - Matching Engine（撮合引擎）
  - Settlement（结算服务）
  - SDK（客户端库）
- 模块间交互流程
- 技术实现细节

**推荐阅读顺序**：了解整体架构和各个模块的功能。

### 3. [路由与撮合引擎工作原理](routing-and-matching-zh.md)

深入讲解路由模块和撮合引擎的实现细节，包含：

- 路由模块详解
  - 市场到撮合引擎映射
  - gRPC 客户端管理
  - 订单格式转换
- 撮合引擎详解
  - 订单簿数据结构
  - 撮合算法详解
  - 价格-时间优先级规则
- 撮合示例和场景分析
- 性能优化策略

**推荐阅读顺序**：深入了解路由和撮合的核心实现。
